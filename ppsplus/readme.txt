The Plus Component of PhyloPythiaS+ (the algbioi package)

REQUIREMENTS
To run the python scripts, we require: 
python 2.7.3
biopython 1.64
The package is dependent on the tools and other files provided in the PPS+ distribution.


PPS+ INTEGRATION
The package is used in the PPS+ distribution and located in:
/apps/pps/tools/ppsplus/algbioi


HOW TO RUN THE PIPELINE
Please consult the tutorial contained in the PPS+ distribution.


DOCUMENTATION:
Gregor et al. (2016) PhyloPythiaS+: a self-training method for the rapid reconstruction of low-ranking taxonomic bins from metagenomes. PeerJ. 
(https://peerj.com/articles/1603/)
